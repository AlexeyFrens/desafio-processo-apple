# RPG - Jogo de Aventura

Este é um jogo de aventura baseado em texto desenvolvido em Swift. O jogador assume o papel de um personagem que deve navegar por um cenário de ficção científica/horror, enfrentar monstros, coletar itens e resolver mistérios. A história se desenrola por meio de escolhas que afetam o progresso do jogo.

## Como Rodar o Jogo

### Requisitos

- **Swift**: Este jogo foi desenvolvido usando a linguagem Swift.
- **Xcode** (recomendado): Embora o código possa ser compilado via terminal, o Xcode oferece uma experiência mais completa.

### Instruções

1. Clone o repositório para o seu computador.
2. Abra o projeto no Xcode.
3. Compile e execute o código.

## Funcionalidades

- **Exploração**: O jogo apresenta várias cenas e ambientes para o jogador explorar.
- **Batalhas**: Enfrente monstros com diferentes níveis de dificuldade.
- **Itens**: O jogador pode coletar itens para melhorar suas habilidades.
- **Progressão**: O jogo apresenta diferentes estados e cenas com base nas escolhas feitas pelo jogador.

## Como Jogar

1. **Início do Jogo**: O jogo começa em uma biblioteca, onde o jogador acorda após desmaiar.
2. **Escolhas**: Em cada cena, você terá opções para escolher, o que influenciará o andamento da história.
3. **Batalhas**: Se o jogador tiver força, pode batalhar contra os monstros. A batalha é automática.
4. **Itens**: Ao explorar os cenários, o jogador pode encontrar itens que aumentam sua força.
